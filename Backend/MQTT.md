#### 主题 - Topic
1. 每个主题至少包含一个字符；
2. 区分大小写；
3. 允许空格；
4. 单独的正斜杠也是一个有效的主题；

##### 通配符 - wildcard
只能用来订阅主题，不能用于发布消息；
- 单级 - signle-level + 
	a/+/b  +可以匹配到任意字符
- 多级 - multi-level  #
	只能放在主题末尾；
	a/b/# 可以匹配到b后的任意层级；
	a/b/c,a/b/d - true
	a/c/c ,b/b/c - false

##### QOS
1.最多一次(0)  
	单向流动
2.至少一次(1)
	双向流动
3.恰好一次(2)
	四次握手
